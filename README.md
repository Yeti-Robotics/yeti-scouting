# Yeti Scouting

## Setup
- Install Java JDK
- Install VSCode with the Java Extensions
- Install NodeJs

## Running Locally
### API
Run the Gradle `bootRun` task
- Mac:  `./gradlew bootRun`
- Windows: `./gradlew.bat bootRun`

#### Local database
To view the database visit: 
http://localhost:8080/h2
- username: scouting
- password: scouting
